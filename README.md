# Tugas Makalah IF2123 Aljabar Linear dan Geometri - Analysis of Homography Transformation Application in Document Scanning Software 
## HOW TO RUN
### Setup
Install seluruh requirement dari python pada terminal anda, disarankan untuk menggunakan python virtual environment
```bash
python3 -m venv venv
source venv/bin/activate
pip3 install opencv-python
pip3 install numpy
```
Pastikan gambar yang ingin di-scan terdapat pada directory yang sama, kemudian jalankan command berikut
```bash
python3 DocScanner.py
```
